# todo-app-vanilla-js
A beginner-friendly To-Do Application built with HTML, CSS, and JavaScript. Allows users to add, edit, delete, and mark tasks as complete, with persistent storage using local storage.

### Features
- Add, edit, delete tasks
- Mark tasks as completed
- Prevent duplicate tasks
- Data persists using localStorage

### Tech Stack
- HTML
- CSS
- JavaScript (DOM manipulation)

### What I Learned
- DOM events and state handling
- localStorage usage
- Writing cleaner JS functions
